# Keyboard Shortcuts

## Global Shortcuts

|      Input       | Description                             |
|:----------------:|:----------------------------------------|
|     Ctrl + S     | Save Project                            |
| Ctrl + Shift + S | Save Project As                         |
|     Ctrl + Z     | Undo                                    |
|     Ctrl + Y     | Redo                                    |
|     Ctrl + Y     | Redo                                    |
|       ESC        | Deselect any currently selected objects |

## Viewport Shortcuts

|  Input   | Description                                                                        |
|:--------:|:-----------------------------------------------------------------------------------|
|    F     | Frame Selection                                                                    |
|    Q     | Selection Tool                                                                     |
|    W     | Move Tool                                                                          |
|    E     | Rotation Tool                                                                      |
|    R     | Scale Tool                                                                         |
|    S     | Snapping Tool                                                                      |
| Ctrl + T | Teleport Tool - Bring the currently selected object to the point under the cursor. |
